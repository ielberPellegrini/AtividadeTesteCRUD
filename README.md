
## FOLHA DE PONTO


Clone o projeto

```bash
  git clone git@github.com:ielberPellegrini/AtividadeTesteCRUD.git
```

Entre no diretório do projeto

```bash
  cd AtividadeTesteCRUD
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
